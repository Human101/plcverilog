## Installing
### Requirements
- C++ compiler that supports C++11
- CMake 2.8 or above

### Testing
- Boost.Test

```console
$ git clone https://github.com/NigoroJr/hashigo
$ cd hashigo/build
$ cmake ..
$ make
$ ./src/hashigo <L5X file>
```
